# Backend Spring Boot

Ce dossier contient le backend principal de CRM Manara.

## Contenu

- code Java Spring Boot
- configuration sécurité
- services métier
- endpoints REST `/api`
- vues Thymeleaf héritées de la V1
- tests Maven
- scripts SQL de référence

## Commandes

```bash
./mvnw spring-boot:run
./mvnw test
```
